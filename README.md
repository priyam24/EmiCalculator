# Author: Priyam Roy

Overview: This is a frontend application using Angular for calculating loan emi by accepting the parameters using a form and call the backend 
service to calculate the emi amount for the given inputs and display the result or show appropriate error message.

Steps to run this project in local:
1. Install node v22.19.0
2. Install any of angular 20
3. Import this repo
4. Run npm install to download the packages
5. Run ng serve to run this application and access this url in the browser: http://localhost:4200
